
# MS-WORKER

Projeto auto escalável construido com spring cloud.

## Tecnologias utilizadas
- Spring cloud
- h2 Database
- Eureka server 
- Api gateway zuul
- Spring cloud security com token jwt
- Servidor de configurações remoto via repositorio git
- openfeign Client Rest
